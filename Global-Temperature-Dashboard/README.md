# 🌍 Global Temperature Dashboard

An interactive web application that visualizes how global temperatures have changed over time, built using Python, Pandas, and Streamlit.

## 📋 Project Overview

This project takes real-world global temperature data (monthly averages from 1850 to today), cleans it, and displays it through a simple, interactive dashboard. Anyone can open the app, pick a range of years, and instantly see how the temperature changed during that period — no coding knowledge required to use it.

## 🎯 What This Project Does

- Loads real climate data from a public source
- Cleans the data so it's ready to work with
- Displays the data in an easy-to-use web app
- Lets the user choose which years to explore
- Shows a chart that updates automatically based on the user's choice

## 🛠️ Technologies Used
